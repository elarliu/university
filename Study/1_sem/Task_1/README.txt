ФАЙЛ 1.с

        Дает ответ на вопрос о том, каким образом "малое" целое расширяется до int:
    старшие разряды int заполняются нулями или знаковым разрядом "малого" целого.
    
        Ввод данных не подразумевается, в качестве результата программа выводит значение двух переменных типа int,
    в которые были явно присвоены значения двух других переменных типа short (со значениями 1 и -1 соответсвенно), 
    а также их битовую запись.

ФАЙЛ 2.с

        Дает ответ на вопрос о том, каким образом при выполнении операции присваивания и явном приведении происходит
    преобразование беззнаковых целых (M-битовое представление) к беззнаковым целым (N-битовое представление) 
    при M > N, M = N, M < N.
    
        Ввод данных не подразумевается, в качестве результата программа выводит значение переменной типа int после 
    присваивания и явного приведения к переменной типа long, а также значение переменной типа long после присваивания
    и явного приведения к переменной типа int (значения переменных одного типа целиком помещаются друг в друга).
